# 宝塔面板 · 纯净版说明

本仓库在官方 11.8.1 源码基础上做了**使用体验精简**，目标：

- **免绑定**官网账号即可使用面板
- **无付费广告 / 无推荐推广 / 无续费弹窗文案**
- **简洁**：不因账号状态跳封锁页

## 已做改动（摘要）

| 能力 | 行为 |
|------|------|
| 官网绑定 | `public.is_bind()` 恒为 True；`/bind` 回首页 |
| 协议页 | 自动写入 `data/licenes.pl`，跳过 license 页 |
| 账号封锁 | `is_error_path()` 不跳转；云端账号状态检查空实现 |
| 付费推荐 | `get_pay_type` / `GetAd` 等返回空 |
| 升级文案 | `get_pd` 不返回续费 HTML |
| 推荐/问卷 | 软件推荐、NPS、活动拉新关闭 |
| 体验改善计划 | `get_improvement()` 恒 False；设置页开关移除；接口拒绝开启 |
| 面板评价 | `show_evaluate` 固定关；NPS 拉取/提交接口 stub |

## 未做（刻意）

- **不**伪造专业版/企业版授权  
- **不**绕过付费插件安装与运行校验（`PluginLoader` 仍在）  
- 免费功能、站点/数据库/文件等正常使用  

## 部署

将 `panel/` 同步到服务器 `/www/server/panel/` 后：

```bash
bt restart
```

浏览器强制刷新（Ctrl+F5）。

## 标记文件

`data/pure_mode.pl` 表示当前为纯净版配置；  
`data/hide_ad.pl` 等开关与接口层固定逻辑双重保证无广告。
